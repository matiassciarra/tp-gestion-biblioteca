# Gestion de Libros de Biblioteca

## Como installar proyecto

Entra a la carpeta backend y ejecuta

```bash
pnpm install
```

o usa

```bash
npm install
```

### Como abrir proyecto

Para abrir server ejecuta

```bash
npm start
```
